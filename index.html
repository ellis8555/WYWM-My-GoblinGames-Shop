Lonnie's project notes!

Here's a couple of things I'd like to point out.

  1. I used Node and Webpack for use with Firebase. I chose not to CDN through Script tag.
  2. Firebase, email scripts and async/await usage are located in /resources/scripts/checkout/submit/submitOrder.js
  3. The shopping cart accepts items from both the Shop and Products page
  4. The submit order button when checking out becomes active once all fields are verified.
  5. The Shop page fetches goblin card data from '/resources/json/goblinData.json'
  6. The Products page grabs 3 random games from two seperate genres from the games API at 'https://api.rawg.io/api/'.
  7. The 'Products' page has 3 break points. Medium screens and lower have game cards displayed as block and each cards width increased
  8. Small screens each cards width is reduced from that of medium sized screens
